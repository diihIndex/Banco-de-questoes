# Project Documentation

## Title: Banco de Questões

## Description
Banco de Questões is a comprehensive question bank application that allows users to create, manage, and take assessments. It aims to provide educational institutions and teachers with a robust platform to generate tests and quizzes efficiently.

## Features
- User Authentication
- Question Creation and Management
- Assessment Creation
- User Progress Tracking
- Reporting and Analytics

## Installation
1. Clone the repository: `git clone https://github.com/diihIndex/Banco-de-questoes.git`
2. Navigate to the project directory: `cd Banco-de-questoes`
3. Install the required dependencies: `npm install`
4. Start the application: `npm start`

## Usage
- To log in, go to `/login` and enter your credentials.
- You can create new questions in the `/questions` section and manage existing ones.
- Navigate to `/assessments` to create and manage assessments.

## Technologies Used
- JavaScript
- Node.js
- Express
- MongoDB
- React
